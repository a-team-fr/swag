{\bfseries{Font\+Awesome}}

Font\+Awesome is a signleton object. Once it is referred in your code, it will load the font into memory. Then it will be available for all other components. Therefore, it is recommended to refer it on main.\+qml.

{\bfseries{Font\+Awesome.\+font\+Family}}

It hold the font name. You may pass to the {\ttfamily font.\+family} in Text component

{\bfseries{Icon Table}}

Moreover, Font\+Awesome object also hold a icon table. For complete icon list, please check the source of \href{https://github.com/benlau/fontawesome.pri/blob/master/FontAwesome/FontAwesome.qml}{\texttt{ Font\+Awesome.\+qml}} file.
